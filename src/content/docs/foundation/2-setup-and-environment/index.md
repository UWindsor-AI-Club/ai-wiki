Before you can start building machine learning models, you need to set up your development environment. This section covers the beginning of what you need.

## What's in this chapter?

- **Installing Python**: Getting Python and setting up a virtual environment
- **Core Libraries**: Installing core libraries for machine learning
- **Activity: First Script**: Running a simple "Hello ML" setup script to verify everything works

## What you will achieve

By the end of this section, you'll have:

- Python installed on your computer
- A virtual environment for isolated project dependencies
- Essential ML libraries ready to use
- Confidence running Python code from the command line

Let's get started!
